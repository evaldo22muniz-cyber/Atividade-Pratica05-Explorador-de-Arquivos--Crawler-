# Atividade-Pratica05-Explorador-de-Arquivos--Crawler-
Cenário: Você precisa criar um utilitário que liste todos os arquivos dentro de uma pasta e, se encontrar subpastas, entre nelas e liste os arquivos de lá também, indefinidamente. Isso é um problema clássico de árvore de diretórios resolvido com recursão.
